EEdfw Geoplugin
===============

Module to return a user's geolocation information via the GeoPlugin service (http://www.geoplugin.com/).

Usage
-----

### {exp:eedfw_geoplugin:locate}

Locates a user.

#### Parameters

ip_address
Specify a users email address. Default: Current user's ip.
